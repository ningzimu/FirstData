## 🌟 收录标准 Inclusion Criteria

1. **学术权威性**: 由知名研究机构、大学或学术组织维护
2. **数据质量**: 数据经过严格的质量控制和验证
3. **可访问性**: 提供公开访问或注册后免费使用
4. **文档完整**: 提供详细的数据说明和方法论文档
5. **引用价值**: 在学术界具有广泛引用和认可
